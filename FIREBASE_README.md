## âš™ï¸ Configuration Steps

### **1. Create .firebaserc**

Copy the template:
```bash
cp .firebaserc.template .firebaserc
```

Edit `.firebaserc` and replace `your-project-id`:
```json
{
  "projects": {
    "default": "meraplion-ambassador"
  }
}
```

### **2. Environment Variables**

Verify `.env` file:
```properties
REACT_APP_GOOGLE_SCRIPT_URL=https://script.google.com/...
REACT_APP_TRACK_PAGE_VIEW=true
REACT_APP_PAGE_VIEW_DELAY=30000
```

### **3. Firebase Project**

Create project at: https://console.firebase.google.com
- Click "Add project"
- Enter project name
- Enable Hosting

## ğŸ”§ Troubleshooting

### **Build Fails**
```bash
rm -rf node_modules package-lock.json
npm install
npm run build
```

### **Deploy Fails**
```bash
firebase logout
firebase login
firebase deploy
```

### **404 on Refresh**
Check `firebase.json` has rewrite rule:
```json
"rewrites": [
  { "source": "**", "destination": "/index.html" }
]
```
